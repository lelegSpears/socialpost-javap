Projeto Post e Comentários em Java:

Este projeto é uma aplicação simples em Java que simula a criação de um Post com Comentários, aplicando conceitos fundamentais de Programação Orientada a Objetos (POO).

O sistema permite:
Criar um post com título e conteúdo, adicionar comentários ao post e exibir todas as informações formatadas no console.

O projeto aplica os seguintes conceitos:

				  
Programação Orientada a Objetos, Encapsulamento, Composição, Uso de StringBuilder e Coleções (List e ArrayList).
